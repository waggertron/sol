# Source Card: Personality Traits and Complex Problem Solving

## Citation

Frontiers in Psychology article linked by the project founder:
https://www.frontiersin.org/journals/psychology/articles/10.3389/fpsyg.2022.788402/full

Source registry id: `frontiers_2022_cps_personality`

## Domain

- personality psychology
- applied psychology
- complex problem solving
- trait-to-behavior modeling

## Key Contribution

The paper is useful because it connects personality traits with complex
problem-solving behavior rather than treating personality as a purely descriptive
label. For this project, the important move is the layering:

- trait constructs
- task or context
- observable behavior
- performance or output

That structure is more useful than trying to infer a single global personality
summary.

## Model Implications

The project should preserve a distinction between:

- underlying trait hypotheses
- task context
- observed behavior
- generated-output preferences
- confidence in each inference

The RAG should retrieve this source when reasoning about whether inferred
personality should directly control output or pass through task/context
mediators.

## Limits and Cautions

This source should not be used to claim that personality can be precisely
decoded from arbitrary user data. It is one empirical anchor for relating
personality to behavior under task conditions.
