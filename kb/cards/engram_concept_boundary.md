# Source Card: Engram Concept Boundary

## Citation

The term "engram" originates in memory research and neuroscience, where it
refers to a physical or functional memory trace. This project uses the term only
as a bounded metaphor unless a neuroscience-specific source is under discussion.

Source registry id: `engram_concept_boundary`

## Domain

- neuroscience terminology
- product metaphor hygiene
- model boundary setting

## Key Contribution

The word "engram" is evocative but risky. It can imply that the system is
capturing a literal trace of the user's mind. That would overstate the project.

## Model Implications

Use "engram" internally to mean:

> an evidence-backed, user-correctable representation of behavioral,
> communicative, motivational, and aesthetic patterns used for personalization.

Public or product-facing terms should usually be:

- profile
- style model
- preference model
- personality and style model
- personalization memory

## Limits and Cautions

Do not use engram language to justify determinism, diagnosis, mind reading, or
claims of neurological validity.
