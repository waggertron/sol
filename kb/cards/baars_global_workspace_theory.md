## Key Contribution

Global Workspace Theory gives a useful architecture for thinking about how
information becomes globally available, reportable, and able to influence other
systems. The important move is not "consciousness" as a product feature. The
useful abstraction is a competition-and-broadcast model:

- many specialized processes run in parallel
- a small subset wins access
- the winning content becomes broadly available
- broad availability changes coordination, memory, and action selection

For this project, that architecture is relevant because it suggests a layered
profile model where user-relevant signals are not treated as one flat trait
vector. Instead, some information should become more globally available to the
generation system only after evidence aggregation and user confirmation.

## Model Implications

Use this cluster to support:

- separation between local observations and globally active profile atoms
- gating of high-impact inferences behind stronger evidence or user approval
- modeling of attention, salience, and current mode separately from stable
  traits
- representation of an "observing self" or self-monitoring layer as a
  user-declared or user-corrected component, not as an inferred hidden truth

This also supports a product distinction between:

- latent background tendencies
- currently active mode or context
- what should be surfaced to generation controls right now

That distinction is directly useful for multimodal personalization. Not every
observed pattern should become an always-on steering variable.

## Limits and Cautions

This theory is a model of conscious access and cognitive coordination, not a
validated recipe for inferring personhood from digital traces.

Do not use it to claim:

- that the system can detect consciousness
- that a user model corresponds to an internal subjective state
- that hidden motives or selfhood can be robustly inferred from sparse data

For this repo, the safe use is architectural and conceptual: it helps structure
how evidence, salience, active mode, and user-visible controls might relate.
